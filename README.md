# praim-test-app
Тестовое задание в компанию "Агентство Экономической Информации ПРАЙМ"

## Установка
Используйте [maven](https://maven.apache.org/download.cgi) для сборки проекта.<br/>
В корневой папке проекта с pom.xml файлом выполните:

```bash
mvn clean package
```
